El documento titulado Teoría HTML ofrece una introducción completa a los fundamentos del lenguaje HTML, su propósito, estructura y las principales etiquetas utilizadas para crear y organizar páginas web.
Comienza explicando que HTML (HyperText Markup Language) es un lenguaje de marcado, no de programación. Esto significa que su función no es realizar acciones o procesar datos, sino estructurar y dar forma al contenido de una página web para que los navegadores puedan interpretarlo correctamente. HTML define la jerarquía y el significado de cada parte del contenido, indicando qué es un título, un párrafo, una imagen, una tabla, un enlace, etcétera.
A continuación, se aclara que las etiquetas HTML son los elementos básicos del lenguaje. Cada etiqueta tiene un formato determinado y se compone de una etiqueta de apertura y una etiqueta de cierre, que encierran el contenido al que afectan. Por ejemplo, Texto indica el inicio y fin de un párrafo. También se menciona que existen etiquetas que no necesitan cierre, como para imágenes o para saltos de línea.
El documento presenta las etiquetas más comunes empleadas en la estructura general de una página. Entre ellas destacan:
html: define el inicio y el fin del documento HTML.
head: contiene información sobre la página que no se muestra directamente en pantalla, como el título o los metadatos.
title: define el título que aparecerá en la pestaña del navegador.
body: contiene todo el contenido visible de la página, como textos, imágenes, tablas o enlaces.
h1 hasta h6: se utilizan para los encabezados o títulos, del más importante al menos importante.
p: se usa para los párrafos de texto.
a: sirve para insertar enlaces o hipervínculos.
img: inserta imágenes en la página.
br: realiza un salto de línea.
Posteriormente se incluyen varios ejemplos prácticos de código, en los que se muestran combinaciones de etiquetas básicas y cómo se visualizan en el navegador. Estos ejemplos permiten ver de manera clara la estructura que sigue un documento HTML y cómo se va construyendo una página con diferentes secciones y tipos de contenido.
En la parte final, el texto introduce el tema de las tablas en HTML, un recurso que permite organizar la información de forma ordenada en filas y columnas. Explica que las tablas se crean mediante una estructura jerárquica de etiquetas:
table: define el inicio y fin de la tabla.
tr: indica una fila dentro de la tabla.
td: define una celda dentro de una fila.
ambién se muestran ejemplos donde se combinan varias filas y columnas para representar datos de manera visual y ordenada. Se entiende que cada fila puede contener varias celdas y que todas ellas forman la estructura completa de la tabla.
El documento finaliza destacando la importancia de HTML como base de toda página web, ya que sin él no sería posible definir el contenido ni su estructura. A partir de HTML, se pueden añadir otros lenguajes como CSS, que se encarga del diseño y la apariencia visual, o JavaScript, que aporta interactividad.
En resumen, Teoría HTML ofrece una explicación clara y progresiva sobre qué es HTML, cómo funciona, cómo se escriben sus etiquetas, cuáles son las más utilizadas y cómo se aplican en ejemplos concretos. Además, introduce el uso de tablas como una forma de organizar información dentro de las páginas, presentando una visión general y sólida de los principios básicos del desarrollo web.
